Settings

The settings are saved under Constants.gs file.

  • Crossover.Url: Base url for XO API
  • Crossover.XAuthToken: The token value for the user. If its set, UserName/Password will not be used.
  • Crossover.UserName: The username for Crossover website
  • Crossover.Password: The password for Crossover website
  • MaxMissingDaysToLoad: Default: 14, Describes how many previous days will be checked for loading on document open. If any day missing within this range, it will load all of the date till yesterday.
  • Crossover.TrackerActivityGroupsTemplate: The value to get metrics from XO API. The default value is "/tracker/activity/groups?date={date}&fullTeam=true&groups=groups&managerId={managerId}&refresh=false&teamId={teamId}&weekly=false". If you want to load only ICs under this user (which is defined by Crossover.UserName), just change fullTeam=false in this url.
